网络状态
手机的网络是否可用，可用则为 “可用”，否则为 “不可用”
网络类型
手机的网络类型，为 “移动网络”，“WiFi”，“其他”
WiFi状态
手机WiFi是否开启，开启则为 “已开启”，否则为 “已关闭”
WiFi信号
手机的WiFi信号，未开启情况下显示 -200dBm，单位为 dBm，信号分类为：无信号、信号差、信号中、信号良、信号优
IP
手机的IP地址，网络状态为 “WiFi” 的情况下为 WiFi IP地址，网络状态为 “移动网络” 的情况下为移动网络 IP 地址
出口IP
手机的外网 IP 地址
DNS
当前配置的DNS解析服务器地址
出口DNS
手机进行 DNS 解析时实际进行迭代查询的 DNS 公网 IP
运营商
手机网络的运营商
网络制式
手机网络的制式，分别为网络制式分类 与 具体网络制式
漫游状态
当前手机移动网络是否正在漫游
手机信号
手机信号，分别为 dBm 与 asu 两种单位的数据，信号分类为：无信号、信号差、信号中、信号良、信号优
