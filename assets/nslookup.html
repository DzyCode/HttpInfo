DNS服务器
手机配置的两个DNS
策略
进行 DNS 解析的策略，分别为：默认策略，指定 DNS 解析
默认策略：如果手机有 DNS 缓存，则使用缓存
指定 DNS 解析：指定 DNS 服务器进行解析，不使用手机 DNS 缓存
HTTPDNS：HTTPDNS 指使用 HTTP 的方式进行 DNS 解析，对应 HTTPDNS 请求链接的主机为 IP 而不是域名，这样在进行 HTTPDNS 请求的时候本身能够绕过 DNS，HTTPDNS 服务为腾讯提供
使用 HTTPDNS 指定用户出口 IP：指使用用户出口 IP 地理位置信息进行解析
使用 HTTPDNS 指定用户出口 DNS：指使用用户出口 DNS 地理位置信息进行解析
域名
被解析的域名
结果
解析域名结果，解析成功则为 “成功”，否则为 “失败”
用时
解析域名所使用的时间，单位为毫秒
IP
解析域名的服务器公网 IP
