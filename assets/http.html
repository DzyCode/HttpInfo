网址
手机发起 HTTP 请求的网址
用时
手机发起 HTTP 请求到返回数据接收完毕总共用时，最多只接收 10KB，单位为毫秒
速度
手机发起 HTTP 请求到返回数据接收完毕的速度，单位为 kbps
下载大小
手机发起 HTTP 请求，接收返回数据的大小，最多只接收 10KB，单位为 KB
服务器
手机发起 HTTP 请求，返回数据 Header 里的 Server 字段
校验服务器
服务器发起 HTTP 请求，返回数据 Header 里的 Server 字段，如果“校验服务器”与“服务器”字段不一致，则存在代理访问，网页可能被劫持
跳转
手机发起 HTTP 请求，该请求是否被重定向
返回Header
手机发起 HTTP 请求，返回数据的 Header，如果存在重定向，则会逐个显示所有请求返回数据的 Header
